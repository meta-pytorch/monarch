Crate generate_api_artifacts 

Source
Expand description

Generates API artifact snapshots for the mesh admin API.

Produces:

  • JSON Schema snapshots (raw schemars output without $id; serve endpoints add $id at runtime per SC-4)
  • OpenAPI 3.1 spec (embeds the JSON Schemas)

Snapshot tests in introspect::tests compare against these files; any type change that alters a schema will fail CI until the artifacts are regenerated.

§Usage

Buck:

buck run fbcode//monarch/hyperactor_mesh:generate_api_artifacts \
  @fbcode//mode/dev-nosan -- \
  fbcode/monarch/hyperactor_mesh/src/testdata

Cargo:

cargo run -p hyperactor_mesh --bin generate_api_artifacts -- \
  hyperactor_mesh/src/testdata

Writes:

  • node_payload_schema.json — NodePayload schema (SC-2)
  • error_schema.json — ApiErrorEnvelope schema (SC-2)
  • openapi.json — OpenAPI 3.1 spec
