pub fn sigpipe_disposition() -> Result<SignalDisposition>
Expand description

Returns the current SignalDisposition of SIGPIPE.

This is a convenience wrapper around query_signal_disposition that checks specifically for the SIGPIPE signal. By default, Rust’s runtime startup code installs SIG_IGN for SIGPIPE (see https://github.com/rust-lang/rust/issues/62569), but this function lets you confirm whether it is currently ignored, set to the default action, or handled by a custom handler.
