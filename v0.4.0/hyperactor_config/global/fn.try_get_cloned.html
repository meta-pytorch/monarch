pub fn try_get_cloned<T: AttrValue>(key: Key<T>) -> Option<T>
Expand description

Try to get a key by cloning the value.

Resolution order: TestOverride -> Env -> Runtime -> File -> ClientOverride -> Default. Returns None if the key has no default and is not set in any layer.

This function reads from a pre-materialized snapshot for lock-free performance.
