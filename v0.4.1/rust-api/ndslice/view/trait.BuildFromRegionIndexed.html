Expand description

Indexed builder: constructs a mesh from sparse (rank, value) pairs.

§Semantics

  • Every rank in 0..region.num_ranks() must be provided at least once.
  • Out-of-bounds ranks (rank >= num_ranks()) cause an error.
  • Missing ranks cause an error.
  • Duplicate ranks are allowed; the last write wins.

§Errors

Returns Self::Error if coverage is incomplete or a rank is out of bounds.
