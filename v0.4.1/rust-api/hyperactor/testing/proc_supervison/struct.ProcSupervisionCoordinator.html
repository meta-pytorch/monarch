Struct ProcSupervisionCoordinator 

Source 
pub struct ProcSupervisionCoordinator { /* private fields */ }
Expand description

Used to create a proc supervison coordinator for testing purposes. Normally you should not use this struct. It is only required in the following cases:

  1. The tests’ logic involves actor failures;
  2. A supervison coordinator is not already set for the proc (e.g. the ProcActor scenario which will be explained later.)

This is because hyperactor’s supervision logic requires actor failures in a proc to be bubbled up to through the supervision chain:

grandchild actor -> child actor -> root actor -> proc supervison coordinator

If the the proc supervison coordinator is not set, supervision will crash the process because it cannot find the coordinator during the “bubbling up”.
