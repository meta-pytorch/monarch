Spawn periodic snapshot capture as a SnapshotCaptureActor.

The actor is spawned on the given proc (same proc as the mesh admin). Lifecycle is framework-managed: proc teardown stops the actor via DrainAndStop. Fire-and-forget — returns ().

cx is any actor context for sending the initial CaptureSnapshot message to the spawned actor.
