Expand description

Remote proc and actor spawn demo.

Run spawner in one process and driver in another. The spawner process starts a UnixProcSpawner, publishes a rendezvous token for its ProcSpawner interface, and waits. The driver joins that token, asks the spawner to spawn a Unix child proc, receives the proc’s SpawnActor interface, spawns a remote Calculator actor through that proc-local spawner, issues one calculation, and exits through Instance::exit. The spawned child process runs the proc role and exits when its actor-spawn endpoint exits. Run the driver as overflow to show a remote calculator failure propagating back to the driver.
