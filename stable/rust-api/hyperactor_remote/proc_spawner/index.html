Remote proc spawning protocol.

ProcSpawnerEndpoint is the proc-level counterpart to crate::ActorSpawnerEndpoint. A proc-spawner endpoint accepts SpawnProc, starts a new proc, creates a proc-local SpawnActor endpoint, and links that actor-spawn endpoint back to the caller through the same remote supervision machinery used for remote actor spawns.
